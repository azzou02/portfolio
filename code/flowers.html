This interactive sketch is inspired by the tension between individuality and conformity through a “superflat” inspired aesthetic. The grid of staring flowers imitates the “Gen Z stare” — a state of detached observation and the anxiety of being watched.

The interactions allow the user to directly manipulate this tension, using color to either isolate individuals or reset the group to a state of collective uniformity.

6 petal design is a nod to the Chinese number for “good luck”, in contrast to the synthetic neon color palette, which feels native to a digital screen rather than a natural garden, reinforcing artificiality.
